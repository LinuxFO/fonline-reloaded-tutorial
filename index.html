FOnline Reloaded Tutorial

Tutorial for everyone who wants to start playing FOnline Reloaded. Step by step instructions.

  • 1. Download the client
  • 2. Configure the client
  • 3. Register character and start playing

1. Download the client

You can download Fallout Online Reloaded's client in this link: https://forum.fonline-reloaded.net/index.php?topic=1.0

Full client is recommended and I'm going to use it in this tutorial.

2. Configure the client

After downloading the FOnline Reloaded's client you should configure it. First of all, run Updater.exe and make sure that host and port are set properly.

Host: europe.fonline.org
Port: 4040

Click Check button and get updates. It might look like this:

Begin checking...
Receive PublicNameColorizing.txt 2470 bytes.
Done, for 0 seconds.
Receive data\reloaded002.zip 74610 bytes.
Done, for 0 seconds.
Update complete.
Checking end.

Now, close Updater.exe and prepare for setting config file. You can change settings in 2 ways:

  1. FOConfig.exe - GUI tool which is responsible for changing different options such as Combat Mode, Resolution or Sound,
  2. FOnline.cfg - configuration file which can be edited by FOConfig.exe or manually by using notepad (example Visual Studio Code).

Let's open FOConfig.exe and check all tabs. You can see my config and do the same way as I usually do.

You can change game's resolution to any resolution as you want to but keep in mind that the game couldn't work properly in higher resolutions. I have tested some of them and you can play in Full HD (1920x1080) without any problems. To change the resolution you have to click input field and write your number (width or height). Remember that you can change resolution (and any other settings via FOnline.cfg too.

After configuring the client, you can run FOnline.exe and register your character.

3. Register character and start playing

Now you can create your first character and start playing FOnline Reloaded.
